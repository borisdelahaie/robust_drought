Robust Drought Analysis Pipeline
Overview

This repository contains the analysis pipeline used to investigate drought responses in Coffea canephora using growth and physiological data.

The workflow combines: Growth-based clustering of genotypes, Longitudinal mixed models of physiological traits, A drought tolerance index (DFI), Statistical comparisons and visualization

Project Structure

├── data_raw/           # Raw input data

├── data_processed/     # Intermediate processed data (e.g. genotype clusters)

├── R/                  # Analysis scripts
